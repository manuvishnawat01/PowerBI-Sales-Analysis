# Power BI Sales Data Analysis

This project analyzes Superstore sales data using Power BI.

## Dashboard Includes

* Sales by Category
* Profit by Region
* Sales by Segment
* Region Filter (Slicer)

## Tools Used

* Power BI
* Superstore Dataset

This dashboard provides insights into sales performance and profitability.
